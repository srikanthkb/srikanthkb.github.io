Crashballs AI Agent

To the previously developed game Crash-Balls, a humble attempt was made towards building an AI agent that plays the game. This AI agent uses Reinforcement Learning to play the game, without being introduced to any rules of the game whatsoever. The goal here was to learn to define states, environment, rewards and learning implementation for an AI agent. The network architecture is built using Keras (High level neural network API). Though the learning curve of the agent is not steep, owing to the randomness and difficulty of the game, I believe further optimization of the parameters of the network model and training for a longer time, will yield better results, towards which I am working on currently. Any ideas, advises to improve this further is appreciated!
